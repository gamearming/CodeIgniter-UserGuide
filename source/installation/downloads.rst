******
GitHub
******

`Git <http://git-scm.com/about>`_ 是一個分佈式版本控制系統。

CodeIgniter 可以在 `GitHub <https://github.com/bcit-ci/CodeIgniter>`_ 上公開存取。請注意，儘管我們在保持程式碼的基礎功能上做出了大量的努力，但是我們並不能為開發分支的程式碼中的功能作擔保。

從 2.0.3 版本開始，每個版本的穩定版都可以從 `GitHub Releases <https://github.com/bcit-ci/CodeIgniter/releases>`_ 讀取。
